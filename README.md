# HealthCheckTool

Log360 Health Check & Product Utilization Scorecard.

A consultant-facing web app to assess a customer's deployment of:

- Log360 (on-prem, 13000-series — UEBA embedded)
- Log360 Cloud
- ADAudit Plus (latest build)
- EventLog Analyzer (13000-series)
- DataSecurity Plus (latest build)

Calculates Health Check (HC) and Product Utilization (PU) scores, generates editable
recommendations, and exports a shareable customer report.
